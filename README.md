# invertTree
Just invert binary tree

## Usage
Install the dependence
```bash
npm install
```
Use default input to invert and show result
```bash
npm start
```

Example
```typescript
input: [5, 3, 8, 1, 7, 2, 6]
output: [5, 8, 3, 6, 2, 7, 1]

input: [2, 1, 3]
output: [2, 3, 1]

input: [4, 2, 7, 1, 3, 6, 9, 100, 3, -1]
output: [4, 7, 2, 9, 6, 3, 1, null, null, null, null, null, -1, 3, 100]

input: []
output: []
```
